# Backend Interview Rubric

## Nhom nang luc can danh gia
- System thinking: hieu luong du lieu, bottleneck, reliability.
- API design: contract ro rang, versioning, error handling.
- Data layer: query efficiency, transaction, consistency.
- Operations: logging, monitoring, incident handling.
- Security basics: authn/authz, input validation, secrets handling.

## Muc do dao sau
1. Clarify: ung vien thuc su dang noi ve dieu gi.
2. Probe: hoi tiep 1 diem cu the trong cau tra loi.
3. Stress: them rang buoc thuc te (budget, scale, deadline).
4. Evidence: yeu cau vi du thuc chien + ket qua do luong.

## Dau hieu cau hoi generic can tranh
- Cau hoi qua rong, khong lien quan boi canh cong ty.
- Cau hoi khong noi tiep duoc cau vua tra loi.
- Cau hoi lap lai y da hoi o turn truoc.
